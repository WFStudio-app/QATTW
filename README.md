> Программа в стиле операционной системы для Web. Окна, терминал, редактор кода, файлы и сайты — всё в браузере.

## Быстрый старт

Просто откройте `index.html` в любом современном браузере — никаких сборщиков, серверов и зависимостей не нужно.

```
qattw/
├── index.html
├── css/
│   └── style.css
├── js/
│   ├── icons.js      — SVG-иконки
│   ├── wm.js         — менеджер окон
│   ├── terminal.js   — содержимое вкладок
│   ├── settings.js   — модальные окна настроек
│   ├── boot.js       — загрузочный экран + canvas-обои
│   └── app.js        — главная логика приложения
└── README.md
```

## Возможности

### Окна

| Действие | Как |
|---|---|
| Создать окно | ⚙ → Создать **или** ПКМ по рабочему столу |
| Переместить | Перетащить за заголовок |
| Изменить размер | Потянуть за правый нижний угол |
| Свернуть | Нажать жёлтую кнопку `─` |
| Развернуть | Нажать зелёную кнопку `□` |
| Закрыть | Нажать красную кнопку `×` |
| Переключить окно | Кликнуть на чип в hot-bar внизу |

### Форматы вкладок

#### Текст
Обычный редактор текста. До **50 000 символов** на вкладку. Внизу — счётчик символов. Вкладки можно добавлять кнопкой `+`.

#### Терминал
Встроенный терминал с поддержкой команд:

| Команда | Описание |
|---|---|
| `help` | список всех команд |
| `ls` | содержимое директории |
| `cd <dir>` | сменить директорию |
| `pwd` | текущий путь |
| `echo <текст>` | вывести текст |
| `date` | дата и время |
| `neofetch` | информация о системе |
| `history` | история команд |
| `clear` | очистить экран |

Стрелки `↑ ↓` — навигация по истории команд.

#### Код
Редактор с подсветкой синтаксиса и запуском кода. Поддерживаемые языки:

- **HLMT** — Hyper Layout Markup Text
- **CSS**
- **JavaScript** (выполняется прямо в браузере)
- **Java**, **C**, **C++**, **Rust**, **C#** (симуляция выполнения)

**Связывание вкладок:** Нажмите «Связать» на вкладке кода, затем кликните на другое окно с кодом. При запуске первой вкладки автоматически запустятся все связанные с ней.

#### Приложения
Раздел зарезервирован. Доступно в версии **v1.1.0**.

#### Сайты
Встроенный браузер. Введите URL или поисковый запрос — откроется в Google, если не указан домен.

#### Файлы
Открытие и просмотр файлов. Поддерживается drag & drop. Текстовые файлы (`.txt`, `.md`, `.js`, `.html`, `.css`) открываются в новом текстовом окне.

### Меню настроек (⚙)

#### Создать
Диалог создания нового окна:
- Выбор иконки-аватарки (PNG, JPG, SVG)
- Название окна (до 32 символов)
- Выбор формата вкладки

#### Настроить
Тонкая настройка внешнего вида через CSS-переменные:
- Цвет акцента
- Цвет текста
- Фон вкладки и окна
- Цвет hot-bar
- Кнопки управления

#### Обои (фон)
- Фото на рабочий стол
- Видео на рабочий стол (autoplay loop)
- Фото в фон вкладок
- Удалить обои (фон → чёрный)

#### Система
Информация о версии, разрешении, платформе и браузере.
Контакт для багов: **wfstudio.wofes@gmail.com**

#### Перезагрузка
Полная перезагрузка с анимацией загрузочного экрана.

---

### Контекстное меню

Правая кнопка мыши по рабочему столу открывает меню:
- Новое текстовое окно
- Новый терминал
- Новое окно кода
- Настройки

---

### Hot-bar

Панель внизу экрана:
- **Часы** (слева) — текущее время
- **Чипы окон** (центр) — переключение между окнами, свёртывание/восстановление
- **Кнопка настроек** (справа)

## Структура кода

```
icons.js        — объект Icons с SVG-строками всех иконок
wm.js           — WM: create, addTab, removeTab, focus, close, minimize, maximize,
                  startLinkMode — полный оконный менеджер
terminal.js     — Terminal: buildText, buildTerminal, buildCode,
                  buildApps, buildSites, buildFiles
settings.js     — Settings: openCreate, openCustomize, openWallpaper, openSystem
boot.js         — startBoot(), initWallpaperCanvas() — частицы + небула на canvas
app.js          — App: init, updateHotbar, toast — основная логика и диспетчер
```

## Известные ограничения (beta)

- Iframe (вкладка «Сайты») блокируется политикой X-Frame-Options большинства сайтов — это ограничение браузера, не баг QATTW
- Данные не сохраняются между сессиями (localStorage — в следующей версии)
- Раздел «Приложения» — заглушка
